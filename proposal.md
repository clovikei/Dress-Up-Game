# Dress-Up Game

## Repository
https://github.com/clovikei/Dress-Up-Game

## Description
A customizable dress up game to create cute characters with a variety of options.

## Features
- Complete customization for different outfits, accessories, and faces on a base. 
	- "Wardrobe" library of assets.
- Clickable UI
	- Switching clothing categories and selecting items
- Sound effects and music
	- Cute music and sound effects on selection.

## Challenges
- Will need to learn how to import images into pygame.
- Create windows and UI in pygame.
- How to layer and stack images so it appears correctly.

## Outcomes
Ideal Outcome:
- Dress up a base character with full customization.

Minimal Viable Outcome:
- Enough variety in assets to create unique designs.

## Milestones

- Week 1
  1. Create window and project structure.
  2. Draw assets.

- Week 2
  1. Code clothing system.
  2. Create UI.

- Week N (Final)
  1. Create backgrounds.
  2. Add sound and polish.
